# Wa-Tor

A simulation of the Wa-Tor predator-prey ecosystem, built with Python and Pygame.

## Setup

### Prerequisites
- Python 3.13+
- pip

### Create the virtual environment
```bash
python3 -m venv watorenv
```

### Activate the virtual environment
```bash
# Linux / macOS
source watorenv/bin/activate

# Windows
watorenv\Scripts\activate
```

### Install dependencies
```bash
pip install pygame
```

### Run the simulation
```bash
python main.py
```
